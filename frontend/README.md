# Frontend — Juventude Tech

Aplicação React + Vite. Veja o [README principal](../README.md) e o
[guia de deploy](../DEPLOY.md) na raiz do projeto para instruções completas
de instalação e publicação.

## Comandos

```bash
npm install
npm run dev      # desenvolvimento (http://localhost:5173)
npm run build    # gera a versão de produção em dist/
npm run lint     # checagem de código
```
